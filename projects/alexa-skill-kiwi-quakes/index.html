Alexa Skill for New Zealand Earthquakes

2018
nodejs • serverless • data

https://github.com/chillu/alexa-skill-kiwi-quakes

New Zealanders often ask themselves: Was that an earthquake, a truck going past, or just the wind? Now you can ask Alexa!

“Alexa, ask kiwi quakes if this was an earthquake”

Under the hood, it uses AWS Lambda functions deployed through the Serverless Framework.

I should acknowledge the awesome New Zealand GeoNet project and its sponsors EQC, GNS Science and LINZ, for providing data used in this app. Check their API and data policy.
